My favorite code snippet

public static int GDC(int num1, int num2) {
    if (num1 == num2) {
        return num1;
    }
    if (num1 > num2) {
        return GDC(num2, num1 - num2);
    }
    if (num1 < num2) {
        return GDC(num1, num2 - num1);
    }
    return 0;
}

It was the first recursive function I learned (its corresponding proof I leanred later was also my first CS proof). I find the implmentation so elegant.
